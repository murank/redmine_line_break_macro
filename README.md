Redmine Line Break macro
=====================================

This plugin allows you to insert a line break anywhere.


Prerequisites
--------------

Tested on Redmine 3.2.1, but it would work on Redmine 2.x.


Installation
------------

1. Clone or download this repository in `plugins` directory.

2. Restart redmine.
